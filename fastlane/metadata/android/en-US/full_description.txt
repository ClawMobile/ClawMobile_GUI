ClawMobile is an on-device field console for preparing and launching the ClawMobile runtime on Android.

* Deploy the ClawMobile base environment from the app.
* Run shell commands directly on the device.
* Start the Ubuntu runtime used by OpenClaw.
* Pair the mobile control bridge and launch the gateway.
* Keep a real terminal available for inspection and recovery.

On first launch, the app prepares the base environment and guides you through the ClawMobile install flow. After setup, you can return to the same device and start the runtime again without repeating the full install.
